Что не удалось выполнить полностью
==================================

1. Не удалось выполнить полный runtime-тест GUI-экспорта изображения.

Причина:
В текущей среде отсутствуют Qt6 development libraries. Проверка конфигурации GUI-сборки выполнялась командой:

cmake -S . -B build_gui_check -G Ninja -DPLOTAPP_BUILD_GUI=ON -DPLOTAPP_BUILD_TESTS=OFF

В `build_gui_check/CMakeCache.txt` зафиксировано:

Qt6_DIR:PATH=Qt6_DIR-NOTFOUND
PLOTAPP_BUILD_GUI:BOOL=ON

Из-за этого GUI-цель приложения в данной среде не была собрана, и открыть окно приложения для ручной проверки экспорта PNG/SVG было невозможно.

2. Не удалось завершить полный прогон Ninja/ctest в песочнице.

Причина:
Полная сборка через Ninja в этой среде несколько раз завершалась прерыванием/таймаутом. Поэтому вместо полного прогона были выполнены сфокусированные регрессионные проверки изменённых зон: XLSX-импорт, повреждённый XLSX, сохранение многострочной легенды и SVG-экспорт многострочной легенды.

Что было проверено вместо этого
===============================

Вручную скомпилированы изменённые core-файлы:

- src/importers/XlsxImporter.cpp
- src/render/SvgRenderer.cpp

Запущен сфокусированный тестовый исполняемый файл, который проверил:

- корректное открытие examples/sample_book.xlsx;
- имя листа Experiment;
- чтение 6 точек из XLSX;
- отклонение повреждённого XLSX исключением без падения процесса;
- сохранение и загрузку многострочной легенды;
- SVG-экспорт многострочной легенды;
- экранирование опасного текста легенды в SVG.

Результат:

Focused tests passed.

Рекомендация для локальной финальной проверки
============================================

На машине с установленным Qt6 выполнить:

cmake -S . -B build -G Ninja -DPLOTAPP_BUILD_GUI=ON -DPLOTAPP_BUILD_TESTS=ON
cmake --build build
ctest --test-dir build --output-on-failure

Затем вручную проверить:

1. открыть приложение;
2. загрузить CSV/TXT/XLSX;
3. открыть экспорт изображения;
4. экспортировать PNG/SVG;
5. создать длинную легенду;
6. создать легенду с переносами строк через Enter;
7. сохранить и снова открыть проект.
