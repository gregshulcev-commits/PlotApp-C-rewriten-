# Изменения в текущем обновлении

## Исправления
- `Role` больше не считается глобальным атрибутом точки.
  - Колонка **Role** показывается только для слоя, созданного плагином `local_extrema`.
  - Окраска точек/легенды по `Role` применяется только к слою min/max.
- Исправлена проблема с `exp(x)`:
  - диалог формулы теперь по умолчанию берет текущий видимый диапазон `X`, если в проекте уже есть данные;
  - добавление формулы в существующий проект больше не делает принудительный сброс viewport, поэтому большие значения `Y` не «схлопывают» остальные графики сразу после добавления.
- Исправлен приоритет операций в формулах: `-x^2` теперь трактуется как `-(x^2)`.
- Исправлен расчет границ графика с учетом error bars в canvas и SVG.

## Выделение и плагины
- Плагин теперь применяется только к **одному выбранному слою**.
- При выборе слоя в дереве слоев выделяется весь слой:
  - для обычного слоя — все его точки;
  - для формулы — вся формула/весь слой.
- Добавлено прямоугольное выделение точек на полотне:
  - сначала выбирается слой в дереве;
  - затем `Shift + ЛКМ + протяжка` выделяет только точки выбранного слоя, попавшие в прямоугольник.
- При запуске плагина в него передается только выделенное подмножество точек выбранного слоя.
- Производный слой сохраняет информацию о том, по каким исходным точкам он был построен.
  Это позволяет после сохранения/открытия проекта и повторного пересчета сохранить тот же самый входной набор точек.

## Архитектура
- ABI плагинов не менялся.
- Поддержка выделения реализована в ядре через передачу плагину уже отфильтрованного представления исходного слоя.
- В модель и сериализацию проекта добавлены `pluginSourcePointIndices`.

## Что проверить на реальной Qt-сборке
- интерактивное выделение прямоугольником;
- повторный выбор того же слоя в дереве как способ вернуть выделение на весь слой;
- сценарий добавления `exp(x)` поверх уже загруженных данных;
- отображение/скрытие колонки `Role` в редакторе точек.
